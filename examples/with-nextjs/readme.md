# Example: Next.js

It uses [`Next.js`](https://github.com/zeit/next.js) a Framework for server-rendered or statically-exported React apps.

## Setup
```sh
$ npm install
$ npm run build
$ npm start
```
or
```sh
$ npm install
$ npm run dev
```

## Usage
Go to `localhost:3000` after starting the server and get the welcome message
